# Краткое описание
Создала с нуля дашборд в Yandex DataLens отделу PR для отслеживания публикаций в медиаполе и выполнение плана

# Контекст и цель
В одном окне отслеживать публикации, и принимать решения на основе этих данных
Все данные в одном окне

# Инструменты и технологии
Python, Google Sheets, API, Git, PostgreSQL, SQL, Yandex DataLens
